# online-film-data-catalog
该仓库用于建立影视相关网站的基础信息目录，对站点定位、栏目构成、页面标题写法及关键词密度分布进行记录说明，便于后续进行SEO结构对比与页面优化研究。仓库不存储任何影视资源，仅整理公开页面中的信息元素。所有资料均来源于互联网公开访问内容，仅作信息索引与研究参考使用，不构成推荐或商业引导。
